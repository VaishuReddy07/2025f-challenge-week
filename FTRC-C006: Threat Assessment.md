## FTRC-C006: Threat Assessment 

### 1. Weak Authentication
Single API key → no user separation  
Impact: Full system access  

---

### 2. API Key Exposure
Key can be leaked or intercepted  
Impact: Account takeover  

---

### 3. Lack of Logging
No tracking of actions  
Impact: No accountability  

---

### 4. Input-Based Attacks
JSON, query, and payload inputs  
Impact: Data corruption, abuse  

---

### 5. DoS Attacks
Rate limiting bypass possible  
Impact: Service disruption
